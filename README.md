# LiftLog 🏋️‍♂️📊

Una aplicación web móvil-first para registrar tus entrenamientos de fuerza de forma 100% privada (los datos nunca salen de tu navegador).

## 🛠️ Tecnologías
- **Frontend**: HTML5, CSS3, JavaScript Vanilla.
- **Almacenamiento Local**: IndexedDB mediante **Dexie.js**.
- **Gráficos**: **Chart.js** (Volumen, 1RM estimado por ejercicio).

## 🚀 Despliegue en GitHub Pages
La aplicación se publica automáticamente mediante **GitHub Actions** en cada push:
1. Sube este código a tu repositorio de GitHub.
2. Ve a **Settings** -> **Pages** en tu repositorio de GitHub.
3. En la sección **Build and deployment** -> **Source**, selecciona **GitHub Actions**.

## 💻 Desarrollo Local
Haz doble clic en [index.html](file:///index.html) para abrirlo en cualquier navegador web.
